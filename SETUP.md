# 구글 시트 연동 설정 (한 번만 하면 됩니다)

이 설정을 마치면 입력한 내용이 자동으로 구글 시트에 저장되어,
**집·회사 등 어떤 컴퓨터에서 열어도 같은 데이터로 이어서 작업**할 수 있습니다.

소요 시간: 약 5분. 코딩 지식 없이 복사·붙여넣기만 하면 됩니다.

---

## 1단계 — 구글 시트 만들기

1. https://sheets.google.com 접속 → **빈 스프레드시트**를 새로 만듭니다.
2. 이름을 알아보기 쉽게 바꿉니다. 예: `주간 업무 기록 데이터`

## 2단계 — Apps Script 열기

1. 그 시트 상단 메뉴에서 **확장 프로그램(Extensions) → Apps Script** 클릭.
2. 새 탭에 코드 편집기가 열립니다. 기본으로 있는 `function myFunction() {}` 내용을 **모두 지웁니다.**

## 3단계 — 코드 붙여넣기

1. 이 저장소의 [`apps-script/Code.gs`](apps-script/Code.gs) 파일 내용을 **전체 복사**합니다.
2. 방금 비운 편집기에 붙여넣습니다.
3. 상단의 💾 **저장** 아이콘(또는 Ctrl/Cmd+S)을 눌러 저장합니다.

## 4단계 — 웹 앱으로 배포

1. 편집기 오른쪽 위 **배포(Deploy) → 새 배포(New deployment)** 클릭.
2. 왼쪽 **⚙️(톱니바퀴) → 웹 앱(Web app)** 선택.
3. 아래처럼 설정합니다.
   - **설명(Description)**: 아무거나 (예: `weeklylog`)
   - **다음 사용자로 실행(Execute as)**: **나(Me)**
   - **액세스 권한이 있는 사용자(Who has access)**: **모든 사용자(Anyone)**  ← ⚠️ 꼭 이걸로!
4. **배포(Deploy)** 클릭.
5. 처음이면 **권한 승인** 창이 뜹니다.
   - 본인 구글 계정 선택 → "이 앱은 Google에서 확인하지 않았습니다" 화면이 나오면
     **고급(Advanced) → (안전하지 않은 페이지로) 이동** → **허용(Allow)** 을 누릅니다.
     (본인이 만든 스크립트라 안전합니다.)
6. 완료되면 **웹 앱 URL**이 나옵니다. 형식:
   `https://script.google.com/macros/s/AKfy............/exec`
   이 URL을 **복사**해 둡니다.

> 다시 확인하려면: **배포 → 배포 관리(Manage deployments)** 에서 URL을 볼 수 있습니다.

## 5단계 — 앱에 URL 연결

방법 A — 앱에서 직접 (가장 쉬움)

1. 배포된 앱 https://miracle03945-eng.github.io/weeklytodolist/ 를 엽니다.
2. 제목 아래 **`연동 안 됨 · 눌러서 설정`** 버튼을 클릭합니다.
3. 4단계에서 복사한 **웹 앱 URL**을 붙여넣고 확인합니다.
4. 상태가 **`저장됨 · 연동 중`** 으로 바뀌면 성공입니다. 🎉

> 방법 A는 그 브라우저에만 저장됩니다. 새 컴퓨터에서 처음 열 때 URL을 한 번 더 붙여넣으면 됩니다.
> 데이터 자체는 어느 기기에서든 자동으로 동기화됩니다.

방법 B — 모든 기기에서 자동 (URL 고정)

매번 URL을 넣기 귀찮다면, URL을 코드에 넣어 고정할 수 있습니다.
[`index.html`](index.html)에서 아래 줄을 찾아 따옴표 안에 URL을 넣고 커밋하세요.

```js
const CONFIG_URL = "";   // ← 여기에 웹 앱 URL을 붙여넣기
```

→ `const CONFIG_URL = "https://script.google.com/macros/s/AKfy..../exec";`

이렇게 하면 어떤 컴퓨터에서 열어도 설정 없이 바로 연동됩니다.
(원하시면 이 URL을 저에게 알려주시면 대신 넣어 배포해 드립니다.)

---

## 확인 방법

- 앱에서 할 일이나 업무를 하나 추가해 보세요.
- 구글 시트로 돌아가면 **`기록`**, **`할일`** 탭이 자동으로 생기고 내용이 채워집니다.
- 다른 컴퓨터(또는 시크릿 창)에서 같은 앱 주소를 열면 같은 데이터가 보입니다.

## 자주 묻는 질문

- **`_state` 탭은 뭔가요?** 앱이 쓰는 원본 데이터입니다. 지우거나 수정하지 마세요.
  보기용은 `기록`·`할일` 탭입니다.
- **여러 컴퓨터에서 동시에 편집하면?** 혼자 쓰는 도구 기준으로 만들어져 **마지막 저장이 우선**됩니다.
  가능하면 한 곳에서 편집을 마치고 다른 곳을 여세요.
- **코드를 고쳤어요.** 배포 → 배포 관리 → 연필(수정) → 버전을 **새 버전(New version)** 으로
  다시 배포해야 반영됩니다. (URL은 그대로 유지됩니다.)
