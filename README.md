# 주간 업무 기록 (Weekly Work Log)

주간 단위로 **할 일을 계획**하고 **한 일을 기록**하는 가벼운 웹 앱입니다.
설치 없이 브라우저에서 바로 쓰고, 구글 시트와 연동하면 어떤 컴퓨터에서든 이어서 작업할 수 있습니다.

**바로 사용:** https://miracle03945-eng.github.io/weeklytodolist/

## 기능
- 주별 할 일 목록 + 진행률, 드래그해서 그날의 업무 기록으로 전환
- 요일별 업무 기록(카테고리·소요시간·상태), 주간 요약 통계
- 구글 시트로 복사 / CSV 다운로드
- **구글 시트 자동 연동** — 여러 컴퓨터에서 같은 데이터로 작업 (아래 설정)

## 구글 시트 연동
어떤 컴퓨터에서 열어도 데이터가 이어지도록 하려면 한 번만 설정하면 됩니다.
👉 **[SETUP.md](SETUP.md)** 의 5분 안내를 따라 하세요.

연동을 설정하지 않아도 앱은 그대로 동작하며, 최근 데이터는 브라우저에 자동 저장(localStorage)됩니다.

## 구성
- [`index.html`](index.html) — 앱 전체 (단일 파일, 정적 페이지)
- [`apps-script/Code.gs`](apps-script/Code.gs) — 구글 시트 연동 백엔드(Apps Script)
- [`SETUP.md`](SETUP.md) — 연동 설정 안내
